# Exam Objectives 1.1

## Categories of security controls:
- Technical:
> **Technology**-based measures such as **firewall and encryption**

- Managerial:
> **Policies**, procedures, and guidlines for security management

- Operational:
> **Day-to-day** security **practices** such as **monitoring and access management**

- Physical:
> Measures to safeguard **physical** **assets** and **premises**

## Types of security controls

- Preventive controls:
> Aimed at **preventing** security incidents

- Deterrent controls:
> Intended to *discourage* potential attackers

- Detective controls:
> Focused on identifying and detecting security incidents

- Corrective controls:
> Implemented after an incident to mitigate the impact

- Compensating controls:
> Alternative measures to *compensate* for **inadequant primary controls**
> Using isolation as a substitute for patch management is an example of a compensating control

- Directive controls:
> Policies or regulations providing specific guidance
