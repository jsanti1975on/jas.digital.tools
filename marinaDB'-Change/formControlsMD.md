# Objects Needed on the Form

| **Control Name**       | **Type**          | **Purpose**                                                      |
|-------------------------|-------------------|------------------------------------------------------------------|
| `txtSlipNumber`         | TextBox          | Enter the slip number.                                           |
| `lblSlipNumber`         | Label            | Label for `txtSlipNumber`.                                       |
| `cmbNoteType`           | ComboBox         | Dropdown for selecting the note type.                           |
| `lblNoteType`           | Label            | Label for `cmbNoteType`.                                         |
| `cmbSpecialist`         | ComboBox         | Dropdown for selecting the specialist.                          |
| `lblSpecialist`         | Label            | Label for `cmbSpecialist`.                                       |
| `cmbSubject`            | ComboBox         | Dropdown for selecting the subject (e.g., "Other").             |
| `lblSubject`            | Label            | Label for `cmbSubject`.                                          |
| `txtNotes`              | TextBox          | Multi-line textbox for entering notes.                          |
| `lblOtherNote`          | Label            | Label for notes when "Other" is selected twice.                 |
| `lblOtherNote1`         | Label (Optional) | Additional label for the advanced mode.                         |
| `cmdOtherNoteSubmit`    | CommandButton    | Button to submit notes when in "Other" advanced mode.           |
| `cmdReset`              | CommandButton    | Button to reset the form to its default state.                  |
| `cmdSubmit`             | CommandButton    | Button to submit the standard form data.                        |
